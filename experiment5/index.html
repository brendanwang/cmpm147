Description

This generator reads inspiring images, renders parameterized images, and adjusts its own designs to optimize an evaluation score.

Technical

  1. This project incorporates a feature where users can select from a variety of inspiring images via a drop-down menu on the display page. These images serve as the basis for initializing the design object in various ways, and depending on the selected inspiration, the initDesign function behaves differently.
  2. The rendered design aims to reflect the chosen inspiration.
  3. The mutation process introduces gradual changes to the design. A key aspect is the implementation of a rate parameter that allows users to control the pace of exploration. By adjusting this, users can effectively slow down exploration, with the option to almost halt it entirely by setting the rate to 1%.

Reflection
