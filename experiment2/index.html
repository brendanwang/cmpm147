I used the p5.js library in order to start my journey in replicating a sunset horizon in some sort of savanna desert. I used various types of random generation and Perlin noise in order to create the scene.

Technical

This experiment represents a dynamic scene that is generated consisting of elements like clouds, stars, mountains, grass, and the sun which has a unique glowing effect. The 'setup()' function initializes the canvas and sets up a button for scene regeneration. When this button is clicked, the scene randomly generates another scene.

The 'draw()' function is responsible for rendering the scene as it begins by resetting the random seed and clearing the background. Then, it draws a gradient sky using linear interpolation between two colors. Stars are also randomly generated across the canvas, each with a randomized size, giving the impression of a night sky.

Next, the sun's glow is drawn using translucent ellipses which are positioned around the sun. Mountain ranges are generated using Perlin noise to create smooth, natural looking peaks. I also positioned the clouds in a way where they travel from left to right of the canvas at varying speeds. I also used a similar technique when generating the blades of grass. I also made them move from left to right, giving the scene a sense of life.

I created initialization functions which set up initial parameters for everything in the scene. The 'reimagineScene()' function then resets these parameters, allowing for the regeneration of a new scene on button press.
